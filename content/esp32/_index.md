## ESP32 + Rust

ESP32 是乐鑫科技推出的低成本 Wi-Fi + 蓝牙物联网芯片，本系列涵盖 ESP32-C3/C6/S3 的 Rust 开发教程。

### 涉及芯片
- ESP32-C3 (RISC-V)
- ESP32-C6 (RISC-V, Wi-Fi 6)
- ESP32-S3 (Xtensa 双核)

### 开发工具
- esp-hal
- Embassy
- espflash
